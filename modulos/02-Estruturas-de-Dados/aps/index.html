Dicionário (Map)

Nossa implementação de lista é definida usando a interface InsperMap, que segue de perto a ADT Map. Você deverá implementar a classe TwoListMap seguindo os algoritmos que estudamos na Aula 02. Essa classe deverá seguir a interface InsperMap.

Atenção

Sua implementação deverá usar variáveis do tipo InsperList para guardar as listas de chaves. Ou seja, sua implementação não pode usar ArrayList, LinkedList ou qualquer outra classe de armazenamento de dados da JVM.

Exercício 1

Vamos fazer agora um contador de palavras em um texto. Dado um texto, seu programa deverá passar por cada palavra e contar o número de ocorrências. Seu trabalho será implementar o método InsperMap<String, Integer> contaPalavras(String texto) da classe br.edu.insper.tecprog.aps01.ContaPalavras.
